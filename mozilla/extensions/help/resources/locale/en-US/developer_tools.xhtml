Web Development Tools

For web developers, &brandShortName; provides several tools to aid in developing and debugging Web applications. Some of these are optional installs. To access these tools, open the Tools menu and choose Web Development.

  • JavaScript Console: the console window displays JavaScript warnings, errors, and notices that occur in Web applications as well as the &brandShortName; application itself.
  • DOM Inspector: a tool that can be used to inspect and edit the DOM of any web document or XUL application.

    Online resources:

    • Introduction to the DOM Inspector (Ian Oeschger)
    • DOM Inspector FAQ (mozilla.org)
    • grayrest's Guide to the DOM Inspector
    • Creating Applications with Mozilla - Appendix B3: The DOM Inspector
  • JavaScript Debugger: also known as Venkman, this utility allows you to debug JavaScript in &brandShortName;. It supports stepping, breakpoints and many other features.

    Online resources:

    • Learning the JavaScript debugger Venkman (Svend Tofte)
    • Debugging JavaScript Using Venkman (webreference.com, article series)
